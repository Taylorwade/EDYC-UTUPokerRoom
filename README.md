EDYC-UTUPokerRoom
=================

Magenta Mondo Matrix Contol Application for DirecTV switching at Horseshoe Poker Room

DirecTV.py
==========

#### Sample usage:
```
py directv.py info 192.168.1.5
```
```
py directv.py channel 192.168.1.5 206
```

Mondo.py
========

#### Sample Usage:

```
#Routing
py mondo.py route [inputNumber] [outputNumber]
```
```
#Prints map
py mondo.py map
```
```
#Saves map to memory
py mondo.py save
```
